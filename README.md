AI Resume Analyzer

AI Resume Analyzer is a Flask-based web application that analyzes resumes using Google Gemini AI. It compares a resume with a job description, calculates an ATS-style score, identifies missing skills, and provides improvement suggestions.

## Live Demo
https://ai-resume-analyzer-cu3m.onrender.com

## Features
- Upload Resume (PDF)
- Paste Job Description
- ATS Score Calculation
- Skill Gap Analysis
- AI-Powered Suggestions
- Resume Matching

## Tech Stack
- Python
- Flask
- Google Gemini AI
- HTML
- CSS
- Bootstrap

## Installation

1. Clone the repository

```bash
git clone <your-github-repo-url>
cd AI-Resume-Analyzer
Install dependencies
pip install -r requirements.txt
Add your Gemini API Key

Create a .env file:

GEMINI_API_KEY=your_api_key
Run the application
python app.py

Open: http://127.0.0.1:5000
